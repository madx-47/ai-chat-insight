## What it is
Utility-first CSS framework. Provides styling for the AI Chat Insight frontend dashboard.

## Key facts
- Version 3.4
- Custom dark theme with zinc base
- Accent colors: cyan and emerald
- Fonts: Space Grotesk + Manrope
- CSS custom properties defined in `frontend/src/index.css`
- `tailwind.config.ts` contains full theme configuration
- `tailwindcss-animate` plugin for animations
- PostCSS for processing
