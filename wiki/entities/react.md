## What it is
JavaScript library for building user interfaces. The frontend dashboard uses React 19 with TypeScript.

## Key facts
- Version 19.2.4
- TypeScript throughout frontend codebase
- Main component: `App.tsx` (634 lines) — contains entire dashboard UI
- Entry point: `frontend/src/main.tsx` (React 18 createRoot API)
- Uses inline SVG for charts (ActiveHoursChart, HeatmapChart, SessionTrendChart, DonutChart)
- Static HTML report embeds React 18 from CDN (not bundled)
