## What it is
Frontend build tool and development server. Bundles the React + TypeScript frontend.

## Key facts
- Version 5.4.8
- Config: `frontend/vite.config.ts`
- Dev command: `npm run frontend:dev` — starts Vite dev server
- Build command: `npm run frontend:build` — produces production bundle
- Uses `@vitejs/plugin-react` for React support
